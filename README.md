# Hangman Game 🎮

A simple command-line Hangman game built using Python. It includes multiple difficulty levels and reads words from a JSON file.

## Features
- Easy, Medium, and Hard modes  
- Words loaded from a JSON file  
- ASCII art for hangman stages  
- Handles invalid input and repeated guesses  

## How to Run

1. Install Python  
2. Clone the repository  
3. Open the folder in terminal or VS Code  
4. Run:

   python main.py

## Project Structure

hangman-game/

├── main.py          # Main game logic  
├── hangman_art.py   # ASCII art and logo  
├── words.json       # Word list
